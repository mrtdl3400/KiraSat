# KiraSat

KiraSat, ASP.NET Core MVC ve SQL Server kullanılarak geliştirilmiş bir kiralama ve satış platformudur. Kullanıcıların ürünleri kiralayabilmesini veya satın alabilmesini sağlayan modern bir web uygulamasıdır.

## Özellikler

* Kullanıcı kayıt ve giriş sistemi
* Rol bazlı kullanıcı yönetimi
* Ürün ekleme, güncelleme ve silme
* Ürün kiralama ve satın alma işlemleri
* Sepet sistemi
* Favori sistemi
* Ürün puanlama sistemi
* Şifremi unuttum özelliği
* Gelişmiş ürün arama ve filtreleme

## Kullanılan Teknolojiler

* C#
* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* Bootstrap

## Proje Durumu

Proje aktif olarak geliştirilmektedir. Yeni özellikler eklenmekte ve mevcut sistemler iyileştirilmektedir.
